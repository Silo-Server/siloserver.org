Silo is designed for native clients and compatibility with parts of the Jellyfin client ecosystem.

## Native clients

The Silo project includes native client work for:

- Apple platforms
- Android
- Web

See the project organization on GitHub for the current client repositories and release status.

## Jellyfin-compatible clients

Silo includes a compatibility layer for clients that speak Jellyfin or Emby-style APIs. Compatibility is evolving, so client behavior can vary by app and by feature area.

Use the Jellyfin-compatible endpoint at `http://localhost:8096` for local testing with compatible clients.
